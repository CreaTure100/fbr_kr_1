## Описание
Это PWA-приложение заметок с поддержкой:
- добавления обычных заметок;
- добавления заметок с напоминанием (дата/время);
- push-уведомлений;
- действий в уведомлении:
   - **Отложить на 10 сек**
   - **Отложить на 5 минут**.

Приложение работает офлайн (через Service Worker и кэширование).

---

## Функциональность

### 1) Заметки
- Сохранение заметок в `localStorage`.
- Удаление заметок из списка.
- Отображение даты/времени напоминания для заметок, где оно задано.

### 2) Напоминания
- В форме можно указать текст и время напоминания (`datetime-local`).
- Напоминание отправляется на сервер, где планируется через `setTimeout`.
- При наступлении времени сервер отправляет push-уведомление подписанным клиентам.



### 3) PWA / Offline
- `manifest.json` для установки приложения.
- `sw.js` кэширует shell и контент страницы.
- Базовая работа интерфейса доступна офлайн.

---

## Стек
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Realtime:** Socket.IO
- **Push:** web-push
- **PWA:** Service Worker + Web App Manifest

---

## Структура проекта

- `index.html` — основной шаблон
- `app.js` — клиентская логика (заметки, reminders, push-подписка)
- `server.js` — сервер, Socket.IO, планировщик напоминаний, endpoints
- `sw.js` — service worker (кэш, push, notification actions)
- `manifest.json` — manifest PWA
- `content/home.html` — интерфейс заметок и формы напоминаний
- `content/about.html` — информация о приложении
- `styles.css` — стили

## Запуск проекта

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск сервера
```bash
npm start
```

Сервер запускается по адресу:
`http://localhost:3001`

### 3. Открыть в браузере
Перейди на:
`http://localhost:3001`
