Запуск практик 4–5

1) Установить зависимости

### Backend
```bash
cd backend
npm install
```

### Frontend
```bash
cd ../frontend
npm install
```

## 2) Запустить backend

```bash
cd ../backend
npm start
```

Backend: http://localhost:3000  
Swagger (практика 5): http://localhost:3000/api-docs

## 3) Запустить frontend (в отдельном терминале)

```bash
cd ../frontend
npm start
```

Frontend: http://localhost:3001
